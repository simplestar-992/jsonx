# JSONX

**CLI JSON Query & Transform Tool**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/jsonx.git
cd jsonx
go build -o jsonx .
```

## Usage

```bash
./jsonx --help
```
